# Starlink Түр Интернэт Нэвтрэх Хандалт Үүсгэх Систем

## Төслийн зорилго

Энэхүү систем нь хэрэглэгчдэд QR код уншуулах замаар Starlink-ийн интернэтэд түр хугацаагаар (1-48 цаг) холбогдох боломжийг олгоно. Хугацаа дуусахад хандалтын эрх автоматаар устах болно. Админы хяналтын самбараар дамжуулан идэвхтэй холболтыг удирдах боломжтой.

## Технологийн сонголт

*   **Frontend:** Next.js (App Router), TailwindCSS, lucide-react
*   **Backend:** Supabase Edge Functions
*   **Database:** Supabase PostgreSQL
*   **Authentication:** Supabase Auth
*   **Scheduler:** Supabase Cron Jobs (`pg_cron`)
*   **Device Integration:** Starlink Local API / SSH Script (Төлөвлөгдөж буй)
*   **QR scanner:** html5-qrcode

## Хөгжүүлэлтийн үе шатууд

### ✔ 1-7-р шатууд (Дууссан)
*   [x] Үндсэн суурь орчин, UI, мэдээллийн сангийн бүтэц, QR уншигч, нэвтрэх эрх үүсгэх болон автоматаар цэвэрлэх функцууд бүгд хийгдэж дууссан.

### 8-р шат: Админ Panel
*   [ ] **Supabase Auth тохируулах:** Админ хэрэглэгч нэвтэрдэг болгох.
    *   [ ] `@supabase/auth-helpers-nextjs` сан суулгах.
    *   [ ] Нэвтрэх хуудас (`/admin/login`) үүсгэх.
    *   [ ] Middleware ашиглан `/admin` хуудсыг хамгаалах.
*   [ ] **Хяналтын самбар (Dashboard) үүсгэх:** `/admin` хуудсанд идэвхтэй холболтуудыг харуулах.
    *   [ ] Идэвхтэй холболтын жагсаалт (SSID, нууц үг, үүссэн цаг).
    *   [ ] Холболт тус бүрд үлдсэн хугацааг тоолох (Real-time countdown).
*   [ ] **Албадан салгах (Force Disconnect) функц:**
    *   [ ] Хэрэглэгчийг албадан салгах товч нэмэх.
    *   [ ] `force-disconnect` нэртэй Edge Function үүсгэж, мэдээллийн сангаас устгадаг болгох.

### 9-р шат: Security & Optimization
*   [ ] **JWT validation:** Edge Function дуудагдах бүрд админ эсэхийг шалгах.
*   [ ] **Rate Limiting:** Нэг IP хаягаас QR код уншуулах хүсэлтэд хязгаарлалт тавих (Upstash зэрэг үйлчилгээ ашиглах).
*   [ ] **Audit Logging:** Админы хийсэн үйлдэл бүрийг (жишээ нь, албадан салгах) тусгай `audit_log` хүснэгтэд тэмдэглэх.
*   [ ] **Notifications (Сонголт):** Холболтын хугацаа дуусахаас өмнө админд мэдэгдэл илгээх.

### 10-р шат: Deploy & Test
*   [ ] **QR Scan Test:** Түгээмэл гар утасны загварууд дээр QR уншигчийн ажиллагааг шалгах.
*   [ ] **Expire Event Test:** Cron job хугацаатай ажиллаж, хуучирсан бичлэгийг зөв устгаж байгааг баталгаажуулах.
*   [ ] **Reconnect Test:** Нэг утас салгаад дахин холбогдоход токен хүчинтэй хэвээр байгаа эсэхийг шалгах.
*   [ ] **Starlink Connectivity Test:** Төхөөрөмж дээр SSID/нууц үг солигдож, устгаж байгаа эсэхийг бодитоор шалгах.
